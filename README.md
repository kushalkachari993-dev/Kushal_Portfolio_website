# Kushal Kachari Portfolio Website

Personal portfolio website for Kushal Kachari, focused on GenAI development, machine learning projects, RAG systems, agentic AI workflows, and research work.

## Overview

This portfolio presents selected AI and full-stack projects, a machine learning research section, skills, background, and contact links. It is built as a Vite + React single-page application and is ready for deployment on Vercel.

## Featured Work

- FinIntel AI: AI-powered Indian financial analysis platform.
- KnowYourRepo: Repository intelligence tool for indexing source code and asking grounded follow-up questions.
- ResumeLink AI: Resume chatbot that generates a public recruiter-facing chat link.
- MediNotes Pro: AI assistant for consultation summaries, action items, and patient communications.
- Personal MCP Server: MCP server deployed on Fly.io for AI assistant workflows.
- Assamese POS Tagger: Machine learning research on Assamese part-of-speech tagging.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/Radix UI components
- React Router
- TanStack Query
- Vitest

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The app runs on:

```txt
http://localhost:8080
```

On Windows PowerShell, use `npm.cmd` if script execution is blocked:

```powershell
npm.cmd run dev
```

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Tests

Run tests:

```bash
npm run test
```

## Deployment

Recommended Vercel settings:

- Framework Preset: Vite
- Install Command: `corepack enable && pnpm install --frozen-lockfile`
- Build Command: `npm run build`
- Output Directory: `dist`
